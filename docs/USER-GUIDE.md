## 2. AI 어시스턴트

가장 강력한 진입점. **자연어 한 줄로 거의 모든 작업** 가능.

### 사용 예시

| 자연어 | 결과 |
|---|---|
| "어제 점심 1만원 카드로" | 가계부 거래 자동 등록 |
| "내일 오후 3시 치과 예약" | 할일 등록 |
| "이번 주 운동 몇 번 했어?" | 통계 답변 |
| "오늘 일정 알려줘" | 오늘의 할일/이벤트 요약 |
| "내일 빈 시간 알려줘" | 일정표 분석 |
| "지난주 식비 얼마 썼어?" | 거래 합계 계산 |
| "퇴근 등록" | 활동 추적 정지 |

### 입력 방법

| 방법 | 단축 |
|---|---|
| ⌨ 텍스트 입력 | 그냥 타이핑 |
| 🎙 음성 입력 | 마이크 아이콘 눌러서 한국어 인식 |
| 📷 영수증 사진 | 클립 아이콘 → 사진 첨부 → "이거 가계부 넣어줘" |
| 📋 클립보드 이미지 | 그냥 붙여넣기 (Ctrl+V) |

### 헤더의 🪙 코인 아이콘

이번 달 AI 호출 비용을 즉시 확인. 기본은 한 달 500원~2000원 수준.

## 3. AI 브리핑

매일 아침/저녁 자동으로 받는 **상황 맞춤 브리핑**.

### 자동 발송 시간 (cron-job.org 설정 시)
- 🌅 **아침 (07:30)**: 오늘 일정/할일 정리 + 격려
- 🌙 **저녁 (22:00)**: 오늘 회고 + 잘했던 점 + 내일 준비

### 받는 방법
1. 홈 화면 **AI 브리핑 카드** → 직접 받기 버튼
2. **PWA 푸시 알림**으로 자동 도착 (cron 설정 시)
3. **🔊 음성으로 듣기** — OpenAI nova 음성 (이모지 자동 제거)

### 비용 절감 팁
- 새로고침은 ~2원 발생하니 **확인 다이얼로그** 통해 의도적으로
- 같은 텍스트 재생은 캐시되어 0원
- 카드 우측 **접기** 버튼으로 화면에서 숨김 가능 (다음날 다시 열림)

## 4. 가계부

### 4-1. 거래 등록 (3가지 방식)

| 방식 | 위치 | 설명 |
|---|---|---|
| 빠른 입력 | 홈/거래내역 우하단 ➕ | 텍스트로 자연어 또는 폼 |
| 음성 | 입력창의 🎙 | "스타벅스 4500원 카드" |
| 영수증 사진 | 입력창의 📷 | 자동 추출 (가게/금액/품목) |

### 4-2. 거래 종류

| 종류 | 용도 |
|---|---|
| 변동지출 | 일상 소비 (식비/교통 등) |
| 고정지출 | 월세·보험·구독료 (자동 알림) |
| 수입 | 월급·이자·환급 |
| 이체 | 계좌 간 자금 이동 (잔액만 변함) |
| 환불 | 결제 취소 |
| 잔액 보정 | 통장 잔고 맞출 때 자동 생성 |

### 4-3. 계좌 관리 (`설정 → 계좌/자산`)

각 계좌 카드 아래 4개 칩:

| 칩 | 동작 |
|---|---|
| 🟢 입금 | 단순 입금 (월급, 환급 등) |
| 🔴 출금 | 단순 출금 (현금 인출 등) |
| 🔵 이체 | 다른 계좌로 송금 |
| 🟣 잔액 맞추기 | 통장 실제 잔액 입력 → 차이는 자동 보정 거래 |

**포인트 계좌** (회사 복지포인트 등):
- 타입에서 **포인트** 선택 → 단위가 P로 표시
- 입금/출금 라벨이 "적립/사용"으로 자동 변경

### 4-4. 카드 청구서 관리 (`설정 → 카드 청구서`)

신용카드 사용 누락 검증.

```
🟦 신한 신용카드 — 11월 청구서
앱 기록 사용액  423,500원
실제 청구액     430,200원  
차이             +6,700원 ⚠️
결제 예정: 12/14 — 생활비통장
```

| 동작 | 효과 |
|---|---|
| ➕ 추가 | 청구기간/결제일/청구액/출금계좌 등록 |
| 결제 완료 처리 | 출금 계좌에서 자동으로 거래 생성 + 잔액 차감 |

### 4-5. 예산 (월별)

`설정 → 예산 설정 → 추가`
- **전체 예산** — 한 달 총지출 한도
- **카테고리별 예산** — 식비 30만, 교통 5만 등
- 사용률이 80% 넘으면 색상 경고

### 4-6. 고정지출 (`설정 → 고정지출`)

월세·보험·구독료 등록 → 매월 등록일에 **홈 화면 알림** 자동 표시.
변동 금액 (관리비 등) 도 체크박스로 처리.

## 5. 할일 / 일정

### 5-1. 종류

| 종류 | 설명 |
|---|---|
| 📌 일정(Event) | 시간 고정 (예: 회의 14:00) |
| 📋 할일(Task) | 기한 있음/없음 (예: 보고서 제출) |
| 🔄 루틴 | Daily Track 으로 별도 관리 |

### 5-2. 입력 방법

`/todo` 우하단 ➕ → **자연어 한 줄**:

```
"내일 오전 10시 치과 예약"
"수요일까지 보고서 제출"
"매주 화요일 7시 운동"
"3주 뒤 정기검진"
```

→ AI가 자동으로 시간/날짜/반복/카테고리 분리.

### 5-3. 화면별 보기

| 경로 | 용도 |
|---|---|
| `/todo` | 오늘 + 미룸 + 곧 다가올 (메인) |
| `/todo/calendar` | 월간 캘린더 |
| `/todo/week` | 주간 시간표 |
| `/todo/day` | 오늘 24시간 슬롯 |
| `/todo/routines` | 루틴 관리 |
| `/todo/archive` | 완료 기록 |

### 5-4. 작업 시간 측정

각 할일 카드 → **▶ 시작** → **⏸ 정지** → 실제 소요시간 자동 기록 → 예상시간과 비교.

자정 넘어가도 자동으로 일자별로 분할 기록됩니다.

## 6. Daily Track (루틴)

매일/주간/월간 반복 트래커.

### 6-1. 등록 (`/todo/tracks`)

```
✅ 12시 전 취침
☑ 매일
🤖 조건: "12시 전 취침"  ← AI 가 활동 시간 보고 자동 체크
```

| 필드 | 효과 |
|---|---|
| 주기 | 매일 / 주 N회 / 월 N회 |
| 조건 텍스트 | AI 가 자연어로 충족 판단 |
| 알림 시간 | PWA 푸시 도착 |
| 종료일 / N회 후 종료 | 한정 도전용 |

### 6-2. 자동 체크 예시

조건이 `"12시 전 취침"` 이라면:
- 활동 추적에서 "수면" 시작이 23:50 → ✅ 자동 충족
- 시작이 01:00 → ❌ 미달

### 6-3. 주/월 트래커 처리

월 1회 트래커가 이미 충족됐다면 **그날만 취소선**으로 보이고 다음날부터 사라집니다. 다음 달 시작되면 다시 보임.

## 7. 활동 추적

수면/공부/운동/독서 같은 **시간 단위 활동** 기록.

### 7-1. 빠른 시작/정지 (홈 화면 칩)

```
[💤 수면] [📚 공부] [💪 운동] [➕ 더보기]
```
- 칩 한 번 탭 → 시작
- 다시 탭 → 정지 (자동으로 시간 기록)
- 자정을 넘기면 자동으로 분할 기록

자주 쓰는 3개가 우선 표시됩니다 (30일 사용 빈도 기준).

### 7-2. 통계 (`/stats`)

- 일/주/월별 활동 시간 그래프
- Top 5 활동
- 평균 수면시간 등

## 8. 아카이브

Notion 같은 컬렉션형 메모.

### 8-1. AI로 컬렉션 만들기

`/archive → 새 컬렉션 → AI 만들기`

```
"읽은 책 정리 — 제목, 저자, 별점, 한줄평"
```

→ AI가 자동으로 속성(컬럼) 설계 → 즉시 사용 가능.

### 8-2. 항목 입력

- AI 자동 채우기 (자연어로 입력)
- 수동 입력
- **Notion 임포트/익스포트** (토큰 등록 필요)

### 8-3. 파일 첨부

이미지/PDF 등 R2에 저장 (선택 — `R2_*` 환경변수 등록 시).

## 9. PWA 설치 + 푸시 알림

### 9-1. 홈 화면에 설치

| 기기 | 방법 |
|---|---|
| Android Chrome | 메뉴 → "홈 화면에 추가" |
| iPhone Safari | 공유 → "홈 화면에 추가" |
| PC Chrome | 주소창 우측 ⊕ 아이콘 |

### 9-2. 푸시 알림 켜기

`설정 → AI 사용량` 위쪽의 **푸시 알림 카드** → **구독** 클릭 → 권한 허용.

받는 알림:
- 🌅 아침 브리핑 (cron 설정 시)
- 🌙 저녁 회고 (cron 설정 시)
- ⏰ Daily Track 리마인더
- 💸 고정지출 등록일 알림

### 9-3. cron 자동화 (선택)

- 가입: https://console.cron-job.org
- 만들 작업 2개:
  - GET `https://본인-앱.vercel.app/api/briefing?mode=morning&push=1` — 매일 07:30
  - GET `https://본인-앱.vercel.app/api/briefing?mode=evening&push=1` — 매일 22:00
- Headers 불필요 / 타임존 Asia/Seoul 자동

## 10. AI 비용 관리

`설정 → AI 사용량` 또는 **AI 어시스턴트 헤더의 🪙 아이콘**.

```
오늘:    35원 (5건)
이번 달: 480원 (62건)
─────────────────
🌅 AI 브리핑    180원 (12건)
🔊 음성 읽기    230원 (40건)
💬 어시스턴트    70원 (10건)
```

### 비용 한눈에
| 기능 | 1회당 |
|---|---|
| 아침/저녁 브리핑 | ~2원 |
| 음성 읽기 (브리핑) | ~5~7원 |
| 어시스턴트 (텍스트) | ~1~3원 |
| 어시스턴트 (이미지) | ~10~30원 |
| 영수증 OCR | ~10~30원 |
| Daily Track 조건 판단 | ~0.5원 |

월 평균 1000~3000원 수준. **OpenAI 대시보드 사용량 한도** 설정 권장 (월 5달러 등).

## 12. 문제 해결 FAQ

### Q. 거래 입력했는데 잔액이 안 맞아요
- 계좌 → **잔액 맞추기** → 통장 실제 잔액 입력 → 차이는 자동 보정 거래로 기록

### Q. 푸시가 안 와요
1. PWA 설치 됐는지 확인
2. 설정 → 푸시 알림 → 구독 상태인지 확인
3. cron-job.org에서 호출이 200으로 성공하는지 확인 (URL은 프로덕션 도메인 — `m-keeper-...` 처럼 git 브랜치 URL 아님)

### Q. AI 어시스턴트가 거래/할일 등록을 못 해요
- OpenAI API 사용량 한도 도달했을 가능성 — OpenAI 대시보드 확인

### Q. AI 사용량이 0건으로 나와요
- `ai_usage` 테이블이 Supabase에 없을 가능성 → SETUP-GUIDE.md의 마이그레이션 SQL 다시 실행

### Q. 친구한테 공유하고 싶어요
- 본인의 데이터는 공유되지 않습니다 (각자 Supabase + Vercel 배포)
- `docs/SETUP-GUIDE.md` 링크를 친구에게 전달하면 됩니다

### Q. AI 브리핑에 이모지가 음성에서 이상하게 읽혀요
- 최근 패치로 자동 제거됩니다. 옛날 캐시 브리핑이라면 새로고침해서 새로 받으세요

### Q. 카드 청구서 결제 완료 처리했는데 거래에 안 보여요
- 거래내역 페이지에서 해당 결제일 확인 — `[카드명] 결제` 라는 이름의 거래가 자동 생성됩니다
- 카테고리는 `카드결제` 로 분류됨
